# Faction Community

Welcome to the official GitHub organization of **Faction Community**.

Faction Community is a messaging and communication platform focused on **privacy, reliability, and security**.

## 🔒 Security

Security is a core priority for us, if you believe you have discovered a security vulnerability, please report it so we can investigate and resolve the issue.

**Please do not report security vulnerabilities through public means, including GitHub Issues, public Faction servers, X (Twitter), Bluesky, or other social media platforms.**

Instead, report them privately:

**Email:** security@faction.chat

When submitting a report, please include:

- A description of the vulnerability  
- Steps to reproduce the issue  
- In your words, the potential impact of the vulnerability  
- Any suggested fixes or mitigations (if you can)

We truely appreciate in depth disclosures and will review all reports as quickly as possible.
